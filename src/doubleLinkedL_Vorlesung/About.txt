
    Abstract Classes
        Se o method foi implementado na classe Abstrata
        nao precisa implementar nas classes que extendem a classe Abstrata
        MAS tu precisa implementar methods of classe abstrata
        se eles nao foram implementados na abstrata.

        Todas classes que EXTENDEM uma classe abstrata compartilham dos
        methods, contrutores e var globais da abstrata
        e precisam implementa-los.

        Se tiver global vars precisa usar super(algo)

        class Klass extends abstract class Abstrata
